Overview

autoSchedule is a family-oriented study planning and reward management app. Key modules: Goals, Dashboard, Rewards, App Lock.

Quick Access
  • Add goal: top-right ＋
  • Open settings: gear icon ⚙︎
  • Change language: Settings → Language
Requirements
  • iOS 15 or later
  • Use a physical device for full features

操作指南

添加目标
  • 点击右上角 ＋ → 选择类型 → 填写名称与日期。
  • 可选：上传 PDF 学习材料，启用 AI 计划生成。
应用锁定（家长密码）
  • 设置 → 应用锁定设置 → 首次将提示创建家长密码。
  • 从列表选择需要锁定的应用或类别后，点击保存并锁定。
奖励中心
  • 完成任务后自动获得奖励分钟数。
  • 在奖励中心查看余额、兑换记录，并设置每日兑换限额。

How-To

Add a Goal
  • Tap top-right ＋ → choose type → fill name and dates.
  • Optional: attach a PDF to enable AI plan generation.
App Lock (Parental Password)
  • Settings → App Lock: you will be prompted to create a parental password on first use.
  • Select apps or categories to lock, then tap Save & Lock.
Reward Center
  • Earn minutes automatically when marking tasks completed.
  • View balance, redemption history, and set daily limits.

故障排查

  • 界面显示本地化键（如 goals.empty.title）：前往设置 → 语言设置，切换一次语言或选择“跟随系统”，然后重启 App。
  • 无法锁定应用：检查“屏幕使用时间”授权是否已开启；如未授权，请在设置页点击“请求授权”。
  • AI 计划生成较慢：PDF 文件过大或网络波动可能导致延迟，请耐心等待或稍后重试。
  • 家长密码忘记：请联系技术支持进行人工重置验证。

Troubleshooting

  • Screen shows localization keys (e.g., goals.empty.title): go to Settings → Language, toggle once or select “Follow System”, then restart the app.
  • Cannot lock apps: ensure Screen Time authorization is granted; tap “Request Permission” in App Lock settings if not.
  • Slow AI plan generation: large PDFs or unstable network may cause delays; please wait or retry later.
  • Forgot parental password: contact support for manual reset verification.

常见问题

  • 如何切换到英文？进入“设置 → 语言设置”，选择 English。
  • 如何删除目标？在目标卡片右上角点击垃圾桶图标并确认。
  • 如何导入图片或PDF？在目标详情页使用“上传/查看图片”，或在添加目标时附加 PDF。

FAQ

  • How to switch to Chinese? Go to Settings → Language and choose Chinese (Simplified).
  • How to delete a goal? Tap the trash icon on the goal card and confirm.
  • How to import images or PDFs? Use “Upload/View Images” in goal details, or attach a PDF when adding a goal.
